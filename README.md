# comp5531-database-internals
Structured revision notes for COMP 5531 (Database Systems) covering storage internals, normalization, indexing, and DBMS design concepts.

This repository contains structured revision notes from COMP 5531 (Files & Databases) at Concordia University.

Topics include relational algebra, normalization theory, physical storage design, indexing structures (B+ trees), and DBMS internals.

The goal is to revisit database systems from a systems-level perspective — connecting logical design with physical storage and query performance.
